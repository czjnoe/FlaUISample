### FlaUI：UI 自动化库
FlaUI 是一个 .NET 库，用于辅助 Windows 应用程序（Win32、WinForms、WPF、应用商店应用等）的自动化 UI 测试。
Github：https://github.com/FlaUI/FlaUI
### FlaUInspect：一个基于 FlaUI 的 UI 自动化检查工具
可以用来查看应用程序的 UI 元素树，属性和模式。
Github：https://github.com/FlaUI/FlaUInspect
